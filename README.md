This is a small Blazor web app rebuilding the birthday market concept.

## Features
- User interface to browse and purchase products (Main Feature)
- Responsive design for mobile and desktop

## Future Enhancements
- Admin panel to manage products, categories, and volunteers (Admin Feature)
- Real-time notifications for order status updates

## Technologies Used
- ASP.NET Core Blazor
- ASP.NET Core Web API 
- ASP.NET Entity Framework Core 
- ASP.NET Core Identity 
- Azure SQL Database 
- SignalR 
- Bootstrap 5
